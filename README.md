# Commit Message 형식

| Header  | Why Use?                                                     |
| ------- | ------------------------------------------------------------ |
| Create  | 완전히 새로운 기능의 추가                                    |
| Fix     | 오류 수정                                                    |
| Remove  | 코드 삭제                                                    |
| Rename  | 이름 변경                                                    |
| Set     | 변수 등의 작은 수정에 사용                                   |
| Update  | 정상 동작의 수정/추가/보완                                   |
| Revise  | 문서 개정                                                    |
| Improve | 정상 동작에서 호환성, 테스트 커버리지, 성능, 검증 기능, 접근성 등의 수정 |
| Move    | 코드 이동                                                    |
